# 云效周报模板

## 📄 标准格式

```
汇报人：XXX
汇报周期：YYYY年MM月DD日 - YYYY年MM月DD日

一、核心成果

【业财一体化】任务项
1. [任务名称] - 完成/完成度X%
2. [任务名称] - 完成

【RPA开发】任务项
1. [任务名称]（完成度80%）

【其他工作】
1. [工作内容]

二、下周期计划

【业财一体化】任务项
1. [任务名称]【紧急】
2. [任务名称]【高】
3. [任务名称]

【其他工作】
1. [工作内容]
```

---

## 🔧 字段说明

### 项目分组
- 【业财一体化】- 主要项目
- 【RPA开发】- RPA相关开发
- 【RPA维护】- RPA维护工作
- 【其他工作】- 日常运维、会议、文档等

### 进度表示
| 状态 | 格式 |
|------|------|
| 已完成 | `- 完成` 或 `（完成度100%）` |
| 进行中 | `（完成度75%/50%/25%）` |
| 设计阶段 | `设计阶段` |
| 开发阶段 | `开发阶段` |

### 优先级标记
只有"紧急"和"高"优先级才显示：
- `【紧急】` - 紧急任务
- `【高】` - 高优先级任务
